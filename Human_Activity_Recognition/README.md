# Human Activity Recognition (HAR)
This project focuses on classifying human physical activities using data recorded by smartphone sensors (Accelerometer and Gyroscope).

### Key HighLights:
- **Accuracy:** 96.5% achieved on test data.
- **Techniques Used:** Pipeline, StandardScaler, LDA (Linear Discriminant Analysis).
- **Models Compared:** Random Forest, SVC, and KNN.
- **Best Model:** SVC (Support Vector Classifier) with RBF kernel.

## Dataset
The model was trained on the HAR dataset, which includes 3-axial linear acceleration and 3-axial angular velocity at a constant rate of 50Hz.
